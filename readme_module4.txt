1. В подвале сайте при нажатии на кнопка "Поддержка" открывается отдельное окно с текстом, в котором можно написать свой вопрос.
    Функция открытия реализована в файле shared.js, само окно в support.html

2. Для формы регистрации /reg.html был написан скрипт reg.js, использующий библиотеку jQuery и проверяющий
    валидации веденных в поля данные. При нажатии кнопки регистрация, данные собираются в объект UserCredentials
    и далее данные из этого объекта выводятся на экране.
    Страница разработана с использованием JQuery.

3. На странице /index.html была добавлена интерактивная карта Греции, на которой в виде синих иконок расположены все памятники Греции.
    При клике на памятник всплывает (с анимацией) маленькое окошечко на котором возможно 4 пункта информации:
        Имя, Описание, Тип, Ссылка на Wiki
        (но не у всех памятников есть все эти пункты)
    При наведении на иконку памятника она меняет цвет(!)
    Данные о памятниках были взяты из скачанного OSM файла (https://download.geofabrik.de/europe/greece.html), для парсинга которого
    был написан python скрипт "parse-monuments.py".
    Так же на карте присутствует JQuery для анимации всплывающего окошка.

4. На странице /cats.html находится рассказ о котах и мышах, на котором есть фон, два текстовых блока и графический блок (картинка).
    При наведении на слово "КОТ" оно исчезает и вместо него появляется маленькое изображение кота с мышью.
    Анимация реализована на JQuery.
